		         1) int

			 3) bool
			 4) complex

		  -----------------------------------------
			1) str
			2) Bytes
			3) ByteArray

		   -----------------------------------------------------------------
			1) list
			2) tuple
		IV) Set Data Types (  Collections Data Types)
		---------------------------------------------------------------------
			1) set
			2) frozenset
		V) Dict Data Type (  Collections Data Types)
		    ---------------------------------------------------------------
		        1) dict
		VI) None data type
		  -----------------------------------
			1) none
